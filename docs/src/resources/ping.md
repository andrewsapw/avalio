# Ping-ресурс

Проверка ресурса осуществляется при помощи ICMP-протокола.

## Конфигурация

Ниже представлен пример конфигурации ресурса:

```toml
[[resources.ping]]
name = 'srv1'
address = '192.168.1.2'
timeout_seconds = 2
```

Конфигурация состоит из следующих настроек:

- `name` - задает уникальное название ресурса
- `address` - ip-адрес ресурса, который нужно проверять
- `timeout_seconds` - таймаут проверки в секундах
